# PhD Application Form (PHP & MySQL)

This is a simple **PhD application form** built using **PHP** and **MySQL**, allowing applicants to submit their details along with a photo. The data is stored in a MySQL database, and uploaded photos are saved in the `uploads/` folder.

---

## **Features**

- Collect applicant details:  
  - Full Name  
  - Email  
  - Phone  
  - Date of Birth  
  - Gender  
  - Qualification  
  - Research Area  
  - Statement of Purpose (SOP)  

- Upload a photo along with the application  
- Stores all data in MySQL  
- Shows a confirmation message with the uploaded photo
